## ImageAiEnhancerApp v1.0.0

Release date: 2026-02-25  
Tag: `v1.0.0`

First public release of ImageAiEnhancerApp.

### Highlights
- Offline-first WPF desktop app on .NET 8
- Single image enhancement workflow with before/after compare slider
- Batch queue processing with per-job status and batch logs
- Basic enhancement mode with upscale, optional denoise, and optional sharpen
- ONNX Runtime AI mode with CPU and optional DirectML GPU provider
- Super-resolution model integration via Models/models.json
- Model manager UI with folder open/reload and provider visibility
- Safety defaults: no overwrite of originals, output suffixing, collision-safe output paths
